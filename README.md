# Partidos politicos
Este projeto foi desenvolvido utilizando React para oferecer uma experiência de teste rápida e eficiente.

##Estrutura do Projeto
Todos os arquivos de código-fonte estão organizados na pasta src.
Instruções de Instalação
Para configurar o ambiente localmente, siga estes passos:

##Clone o repositório:

git clone [<URL_DO_REPOSITORIO>](https://github.com/FilipeSall/Partidos-politicos-no-Brasil.git)
Instale as dependências:

npm install
Inicie o ambiente de desenvolvimento:

npm run dev
