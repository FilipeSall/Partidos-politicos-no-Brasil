# Partidos politicos
Completei o desafio usando o React para concluir mais rápido (como mostra as gravação).

## Estrutura do Projeto
Todos os arquivos de código-fonte estão organizados na pasta src.
Instruções de Instalação
Para configurar o ambiente localmente, siga estes passos:

## Clone o repositório
git clone https://github.com/FilipeSall/Partidos-politicos-no-Brasil.git

## Instale as dependências:
npm install

## Inicie o ambiente de desenvolvimento:
npm run dev
